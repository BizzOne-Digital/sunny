## 🧪 Kaise Test Karein

### Step 1: Server Restart
```bash
npm run dev
```

### Step 2: Admin Panel Kholen
```
URL: http://localhost:3001/admin
Login: admin@dtdogs.ca / Admin@12345
```

### Step 3: Koi Page Edit Karein
```
1. Click "Pages" in sidebar
2. Click "Home" page
3. Hero Heading change karein (e.g., "TEST - Calm care for pets")
4. Click "Save"
5. Message dekhen: "Saved."
```

### Step 4: Frontend Pe Check Karein
```
1. New tab kholen: http://localhost:3001
2. Refresh karein (F5)
3. ✅ Aapka change dikhai dega!
```

### Step 5: Service Edit Karein
```
1. Admin mein "Services" pe jao
2. "Dog Walking" pe click karein
3. Description change karein
4. Click "Save"
5. Frontend: http://localhost:3001/services/dog-walking
6. ✅ Updated description dikhai dega!
```

### Step 6: FAQ Edit Karein
```
1. Admin mein "FAQs" pe jao
2. Koi bhi question pe click karein
3. Answer change karein
4. Click "Save"
5. Frontend: http://localhost:3001/faq
6. ✅ Updated answer dikhai dega!
```
