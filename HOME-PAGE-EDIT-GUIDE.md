## 🧪 Kaise Edit Karein

### Step 1: Admin Panel Kholen
```
URL: http://localhost:3001/admin/pages
Login: admin@dtdogs.ca / Admin@12345
```

### Step 2: Home Page Select Karein
```
1. Left sidebar mein "Home" pe click karein
2. Right side pe poora form dikhai dega
```

### Step 3: Hero Section Edit Karein
```
Form mein scroll karein, aapko milega:
- Hero Eyebrow field
- Hero Heading field  
- Hero Text field
- Primary Button Label field
- Primary Button Link field
- Secondary Button Label field
- Secondary Button Link field
```

### Step 4: Page Sections Edit Karein
```
Form ke neeche "Page Sections" area hoga:

Section 1: Stats Cards
- Edit 4 cards ka text

Section 2: Our Vision
- Edit heading, paragraph, button
- Change image

Section 3: Why Choose Us
- Edit 6 features
- Change 2 side images

Section 4: How Booking Works
- Edit 4 steps ka text

Section 5: Meet Founder
- Edit heading, paragraph, button
- Change founder image

Section 6: Gallery Preview
- Edit heading, button
- Change 3 gallery images

Etc...
```

### Step 5: Save Karein
```
1. Scroll to bottom
2. Click "Save" button
3. Wait for "Saved." message
```

### Step 6: Frontend Check Karein
```
1. Open: http://localhost:3001
2. Press F5 (refresh)
3. ✅ Aapke changes dikhai denge!
```

## 📝 Example Edits

### Hero Heading Change Karna:
```
Admin:
1. Go to admin/pages
2. Click "Home"
3. Find "Hero Heading" field
4. Change from: "Welcome to DTdogs.ca"
5. Change to: "Premium Pet Care in Toronto"
6. Click Save

Frontend:
1. Go to http://localhost:3001
2. Refresh
3. ✅ See: "Premium Pet Care in Toronto"
```

### Stats Card Text Change Karna:
```
Admin:
1. Go to admin/pages → Home
2. Scroll to "Page Sections"
3. Find Stats section
4. Edit Card 1 text
5. Click Save

Frontend:
1. Refresh home page
2. ✅ See updated card text
```

### "Why Choose Us" Feature Add/Edit Karna:
```
Admin:
1. Go to admin/pages → Home
2. Scroll to "Why Choose Us" section
3. Edit any of the 6 features
4. Or add new feature
5. Click Save

Frontend:
1. Refresh home page
2. ✅ See updated features
```

### Image Change Karna:
```
Option 1 - New Image Upload:
1. Go to admin/media
2. Upload new image
3. Copy image URL
4. Go to admin/pages → Home
5. Find image field in section
6. Paste new URL
7. Click Save

Option 2 - Select from Library:
1. admin/pages → Home
2. Image field ke paas "Browse" button
3. Media library se select karein
4. Click Save
```
